Git记录

Sun, Jul 8, 2018

git创建分支并提交到远程

  1. git branch -r [命令查看远端库的分支情况]
  2. git checkout -b name [从已有的分支创建名为name的分支]
  3. git push origin name [推送新的本地分支到远程]

git回退commit版本

  1. git reset –hard HEAD^ [回退到上一个版本]
  2. git reset –hard HEAD^^ [回退到上上个版本]
  3. git reset –hard HEAD~100 [回退到上100个版本]
  4. git reset –hard commitID [回退到指定commit版本]

git恢复commit版本

  1. git reflog [恢复到之前commit的版本]

pull远程分支到本地

  1. git pull origin master [更新master分支代码]

IntelliJ IDEA 对比当前分支代码与远程master分支代码

  1. IDEA右下角选择当前分支，点击Remote Branches下的origin/master,选择Comparing With进行对比

IntelliJ IDEA Merge master到当前分支

  1. IDEA右下角选择当前分支，点击Remote Branches下的origin/master,选择 Merge into Current

合并操作

  1. add与commit合并：git commit -am “合并提交”
