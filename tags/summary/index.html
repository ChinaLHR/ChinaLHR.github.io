2017年度总结

 Posted on January 1, 2018

编程方面 从大二上学期到大三上学期的总结： 大二开始初识Android开发，期间通过阅读书籍，官方教程等,阅读博客以及参考视频学习Android开发基础与开发Android小项目， 利用期末与寒假时间学习并利用RxJava，Retrofit与Glide，通过豆瓣开放API独立开发完成一个豆瓣Material design风格的客户端。 大二下学期开始学习JavaWeb开发，通过阅读书籍，文档与视频的方式，期间通过视频学习并完成了基于SSH的OA项目，基于SSM，Solr， Redis的分布式商城项目。开始系统地学数据结构与算法，参加了蓝桥杯省赛(一轮游)，期间深入学习Java基础，对并发编程，网络编程，NIO， JVM，Java Lamdba，Stream等有了新的认识，深入学习MySql，HTTP与TCP协议，也利用期末到暑假的时间独立设计与开发了一个基于Springboot的社区系统。 大三开始对Spring Framework，分布式服务，RPC，Netty进行深度学习与探索，对自身计算机科学体系的不足开始补充，从数据结构与算法开始， 对计算机网络，计算机组成原理，操作系统，编译原理等一系列的基础知识开启补完计划，希望自己能坚持下去。 技术书籍 Java 《Java并发编程的艺术》 《实战Java高并发程序设计》 《Java8实战》 《Java NIO》 《实战JAVA虚拟机 JVM故障诊断与性能优化》 《Spring技术内幕》 《架构探险 从零开始写javaweb框架》 《深入剖析Tomcat》 《JavaWeb技术内幕》 《Springboot实战》 《Netty权威指南》 C 《C语言程序设计》 SQL 《深入浅出MySQL数据库开发、优化与管理维护第》 《SQL必知必会》 《Redis实战》 Android 《Android编程权威指南》 《第一行代码》 分布式 《分布式服务框架的原理与实现》 《Spring Cloud 微服务实战》 数据结构与算法 《算法 第四版》 《大话数据结构》 《剑指Offer》 计算机网络 《图解HTTP》 《图解TCP/IP》 设计模式 《设计模式的艺术软件开发人员内功修炼之道》 电影&连续剧 关键字：科幻，战争，纪录片 [Read More]
