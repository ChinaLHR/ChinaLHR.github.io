Java、SpringBoot、Spring WebFlux、Spring Cloud Alibaba、Mybatis、SkyWalking、MySql、Kafka等

职责

作为后端开发人员，参与财务中台服务、内部网关迭代需求开发，架构设计，基础设施完善（APM，质量检测，日志系统等）

行动

  • 负责自研网关架构设计，基于微内核架构模式实现高可扩展、插件化的应用；协调推进项目进度，鉴权、动态路由、数据同步核心功能编写；技术大纲、功能模块文档化；上线后可用性达到99.99%
  • 参与财务中台需求迭代开发，进行业务建模，功能设计，基于Spring Cloud Alibaba技术栈实现基础配置、单据服务、算费引擎等功能；提供SDK的接入方式；适配公司多租户架构，开发自定义数据源，封装成Spring Boot Starter内部使用，避免重复造轮子
  • 协助各个业务组产品线内部功能模块解耦合，协调推动基础配置、算费模块接入财务中台服务；通过双引擎替换策略（新数据双写，旧数据同步）对各组业务进行平滑升级
  • 参与中台基础设施完善，SonarQube结合持续集成推进代码质量优化；中台服务接入SkyWalking进行应用性能监控，调用链路追踪；接入ELK进行日志分析

结果

云空间SaaS平台租赁、物业产品线基础配置、算费模块成功接入财务中台服务，解决算费功能响应慢，交付难等问题，服务运行稳定
